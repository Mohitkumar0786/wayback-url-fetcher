# wayback-url-fetcher
Professional Python CLI extracts ALL archived URLs from Internet Archive CDX API for any domain + subdomains. HTTP 200 filtering, auto-deduplication, saves to &lt;domain>_wayback_urls.txt. OSINT, security recon, SEO audits, digital preservation. pip install -r requirements.txt &amp;&amp; python wayback_url_fetcher.py. Pure Python, just 'requests'.
